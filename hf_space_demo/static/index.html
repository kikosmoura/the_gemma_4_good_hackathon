Technical Note for Judges

The primary implementation of Hansen Guard is a mobile application optimized for on-device inference using Gemma 4 with LiteRT. This enables private, offline screening directly on edge devices, with smartphones being the platform we primarily targeted in this project.

To facilitate the evaluation process, this web-based Hugging Face Space was created as a simplified demo. Since LiteRT is heavily optimized for mobile and on-device environments, this web version uses Gemma 4 via llama.cpp (GGUF) to provide a functional browser-based preview.

Additionally, we replaced the live camera input with a curated gallery of real-world cases so judges can immediately test the model’s screening logic and multimodal capabilities without requiring a physical Android device or local environment setup.

Clinical note

This demo supports community screening and does not replace in-person clinical assessment, neurological examination, or medical decision-making.
